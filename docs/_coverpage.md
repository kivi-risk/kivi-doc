# **kivi** 风控建模

> 一体化风控建模

- 覆盖主流的各类指标分箱工具
- 提供自动化分箱与分箱优化流水线
- 支持向前向后多种逐步自动化回归方式
- 便捷的多样本模型评估，以及各类效果评估工具
